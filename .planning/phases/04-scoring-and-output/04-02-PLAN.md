**Part A: Implement `export_results` local function** (OUTP-02, D-13, D-14)

Add to the Phase 4 local function group (before `%[appendix]`). Add `%[text]` annotation:

```
%[text] **export\_results** -- Auto-saves the leaderboard table to CSV and xlsx in
%[text] the configured export directory with timestamped filenames (D-13). Both stepper
%[text] and BLDC teams are included; BLDC Rank column is NaN (D-14). The xlsx write
%[text] is wrapped in try/catch in case the file is locked by Excel.
```

```matlab
function export_results(T, cfg)
```
- Create export dir if missing: `if ~exist(cfg.export_dir, 'dir'), mkdir(cfg.export_dir); end`.
- Generate timestamp using datetime API (not legacy datestr, per Research Pitfall 4): `timestamp = char(datetime('now', 'Format', 'yyyyMMdd_HHmm'))`.
- Build base path: `base = fullfile(cfg.export_dir, sprintf('%s_%s', cfg.export_stem, timestamp))`.
- CSV export: `writetable(T, [base '.csv'])`. Print confirmation: `fprintf('Saved: %s\n', [base '.csv'])`.
- xlsx export with try/catch (Research Pitfall 3):
  ```matlab
  try
      writetable(T, [base '.xlsx']);
      fprintf('Saved: %s\n', [base '.xlsx']);
  catch e
      warning('scorer:xlsxfail', ...
          'xlsx export failed: %s\nClose Excel and retry. CSV was saved.', ...
          e.message);
  end
  ```
- Also save session workspace for post-hoc inspection (Claude's discretion -- low cost, high value):
  ```matlab
  mat_file = [base '.mat'];
  session_data = evalin('base', 'session');
  save(mat_file, 'session_data', 'T');
  fprintf('Saved: %s\n', mat_file);
  ```
  Note: If `evalin('base', 'session')` is fragile in local function context, instead pass `session` as a 3rd argument: `function export_results(T, cfg, session)` and adjust the finalization call accordingly.

**Part B: Implement `plot_score_breakdown` local function** (D-12)

Add to the Phase 4 local function group with `%[text]` annotation:

```
%[text] **plot\_score\_breakdown** -- Creates a grouped bar chart showing the score
%[text] breakdown per team (stepper + BLDC). Three bar groups per team: TimePoints,
%[text] SMAPEPoints, and ParticipationPoint (D-12).
```

```matlab
function plot_score_breakdown(T, cfg)
```
- Build score matrix: `score_matrix = [T.TimePoints, T.SMAPEPoints, T.ParticipationPoint]` (N_teams x 3).
- Create figure: `fig = figure('Name', 'Score Breakdown', 'NumberTitle', 'off')`.
- Plot grouped bars: `b = bar(score_matrix, 'grouped')`.
- Set colors (Claude's discretion on color scheme):
  - `b(1).FaceColor = [0.2, 0.6, 0.9]` (blue for time points)
  - `b(2).FaceColor = [0.9, 0.5, 0.1]` (orange for SMAPE points)
  - `b(3).FaceColor = [0.3, 0.8, 0.3]` (green for participation)
- Set x-axis labels: `set(gca, 'XTickLabel', cellstr(T.Team), 'XTickLabelRotation', 15)`.
- Add y-axis label: `ylabel('Points')`.
- Add title: `title('Competition Score Breakdown')`.
- Add legend: `legend({'Time Points', 'SMAPE Points', 'Participation'}, 'Location', 'northeast')`.
- Add grid: `grid on`.
- Add total points text above each group: for each team `i`, compute `total = T.TotalPoints(i)` and place text annotation above the tallest bar: `text(i, max(score_matrix(i,:)) + 0.2, sprintf('%.1f', total), 'HorizontalAlignment', 'center', 'FontWeight', 'bold')`.

**Function 1: `aggregate_best`** (D-01, D-02, D-03, SCOR-01)
```matlab
function [best_time, best_smape, has_participation] = aggregate_best(team)
```
- Initialize `times = NaN(numel(team.attempts), 1)` and `smapes = NaN(numel(team.attempts), 1)` and `part_flag = false`.
- Loop over `team.attempts`: for each attempt `j`, read `m = team.attempts{j}.metrics`.
  - If `isfield(m, 'swingup_success') && m.swingup_success`, set `times(j) = m.swingup_time`.
  - If `isfield(m, 'smape_eligible') && m.smape_eligible && ~isnan(m.smape)`, set `smapes(j) = m.smape`.
  - If `isfield(m, 'participation') && m.participation`, set `part_flag = true`.
- Compute `best_time = min(times, [], 'omitnan')` and `best_smape = min(smapes, [], 'omitnan')`.
- Guard empty/all-NaN: `if isempty(best_time) || all(isnan(times)), best_time = NaN; end` (same for best_smape).
- Return `has_participation = part_flag`.

**Function 2: `assign_points_dense`** (SCOR-02, SCOR-03, D-04, D-05, D-06)
```matlab
function pts = assign_points_dense(values, point_table)
```
- `values`: [N x 1] double (lower = better for both time and SMAPE). NaN = did not qualify.
- `point_table`: [1st_pts, 2nd_pts, 3rd_pts, 4th_pts, ...].
- Initialize `pts = zeros(N, 1)`.
- Find `valid = ~isnan(values)`. If `~any(valid)`, return zeros.
- Extract `v = values(valid)`.
- Sort ascending: `[sorted_v, order] = sort(v, 'ascend')`.
- Build dense ranks: initialize `ranks = ones(numel(v), 1)`. For `k = 2:numel(sorted_v)`: if `sorted_v(k) == sorted_v(k-1)`, `ranks(k) = ranks(k-1)` (tie = same rank); else `ranks(k) = ranks(k-1) + 1` (dense increment, no gaps).
- Map ranks to points: for each `k`, `rank_pts(k) = point_table(min(ranks(k), numel(point_table)))`. If `ranks(k) > numel(point_table)`, assign 0.
- Unsort: `pts_valid(order) = rank_pts`.
- Write back: `pts(valid) = pts_valid`.
- D-04 sports convention is automatically satisfied: tied teams share the same rank index, which maps to the same (higher) point value.

**Function 3: `score_bldc_smape`** (SCOR-04, D-07)
```matlab
function pts = score_bldc_smape(smape_pct, bands, band_pts)
```
- `bands`: `[40, 80, 120, 160]` (upper-exclusive boundaries).
- `band_pts`: `[4, 3, 2, 1, 0]` (one more element than bands; last = fallback).
- If `isnan(smape_pct)`, return `pts = 0`.
- Default `pts = band_pts(end)` (160+ case).
- Loop `for k = 1:numel(bands)`: if `smape_pct < bands(k)`, `pts = band_pts(k); return; end`.

**Function 4: `compute_leaderboard`** (SCOR-01 through SCOR-06, OUTP-01, D-01 through D-09)
```matlab
function T = compute_leaderboard(session, cfg)
```
- `N = numel(session.teams)`.
- Preallocate: `team_names = {cfg.teams.name}'`, `team_types = {cfg.teams.type}'`, `best_times = NaN(N,1)`, `best_smapes = NaN(N,1)`, `part_flags = false(N,1)`, `time_pts = zeros(N,1)`, `smape_pts = zeros(N,1)`, `part_pts = zeros(N,1)`.
- Loop `for i = 1:N`: call `[best_times(i), best_smapes(i), part_flags(i)] = aggregate_best(session.teams(i))`. Set `part_pts(i) = double(part_flags(i))` (SCOR-05, D-09).
- Build `stepper_mask = strcmp(team_types, 'stepper')`.
- Stepper time ranking: `time_pts(stepper_mask) = assign_points_dense(best_times(stepper_mask), cfg.time_points)` (SCOR-02).
- Stepper SMAPE ranking: `smape_pts(stepper_mask) = assign_points_dense(best_smapes(stepper_mask), cfg.smape_points)` (SCOR-03).
- BLDC scoring: `bldc_mask = strcmp(team_types, 'bldc')`. For each `i = find(bldc_mask)'`: `smape_pts(i) = score_bldc_smape(best_smapes(i), cfg.bldc_smape_bands, cfg.bldc_smape_pts)`.
- Total: `total_pts = time_pts + smape_pts + part_pts` (SCOR-06).
- Stepper ranks on total (descending): build `ranks = NaN(N,1)`. Extract stepper totals, sort descending, assign dense ranks (1=highest total), write back to `ranks(stepper_mask)`. BLDC rank stays NaN (D-08).
- Construct table: `T = table(string(team_names), best_times, best_smapes, time_pts, smape_pts, part_pts, total_pts, ranks, 'VariableNames', {'Team','BestSwingupTime','BestSMAPE','TimePoints','SMAPEPoints','ParticipationPoint','TotalPoints','Rank'})` (OUTP-01).

### Control Design Validation (Controllability & Observability)
Location: `scripts/stepper/RRpendulum_FSFB_controldesign.mlx`

```matlab
sys = ss(A_lin_num, B_lin_num, C_num, 0)
rank(ctrb(A_lin_num, B_lin_num))             % must be 4 to be fully controlable
rank(obsv(A_lin_num, C_num))                 % must be 4 to be observable

K = lqr(sys, Q, R)                           % LQR feedback gain
Kd = lqrd(A_lin_num, B_lin_num, Q, R, Ts)   % LQR feedback gain discrete
ctrpoles = eig(A_lin_num - B_lin_num*K)     % Closed-loop poles (eigenvalues)
```
- **Setup:** Linearize around inverted pendulum equilibrium
- **Assertion:** Rank checks must equal state dimension (4); poles inspected for stability (negative real parts)
- **Manual step:** User verifies pole locations visually are reasonable (10x separation for observer)
